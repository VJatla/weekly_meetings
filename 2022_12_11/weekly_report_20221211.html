3.1. Using mmaction2 library.

  • The plotting tool from the mmaction2 library does not seem to support plotting of validation accuracy and loss.

    • Tool description link 

      # from  tools/analysis/analyze_logs.py line 59
for epoch in epochs:
    iters = log_dict[epoch]['iter']
    if log_dict[epoch]['mode'][-1] == 'val':
	iters = iters[:-1]
    xs.append(np.array(iters) + (epoch - 1) * num_iters_per_epoch)
    ys.append(np.array(log_dict[epoch][metric][:len(iters)]))
  • Hence I wrote my own code to support plotting of Training and Validation accuracy.
  • Also, by editing workflow parameter, mmaction2 library claims to support calcuation of validation loss.

    • Workflow documetation 

      [('train', 1), ('val', 1)]
    • From my understanding I need to rerun the training to get validation loss. May be there is a way to calculate it from saved checkpoints as described at evaluating a metric.
